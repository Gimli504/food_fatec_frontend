Sobre Nós

Este projeto foi desenvolvido pelos alunos da Fatec como parte das atividades acadêmicas. Nosso objetivo é criar uma aplicação que facilite a gestão de pedidos de alimentos, proporcionando uma experiência agradável para usuários e administradores.

Funcionalidades

shopping_cart Gestão de Pedidos

Facilite o gerenciamento dos pedidos de alimentos de forma rápida e eficiente.

 analytics Relatórios

Gere relatórios detalhados para acompanhar o desempenho e as vendas.

 people Perfil de Usuário

Gerencie os perfis dos usuários com permissões e acessos diferenciados.
